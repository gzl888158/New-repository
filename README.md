# 欧易全自动加密货币交易系统
基于多周期MACD共振+ATR动态风控的量化交易系统，适配欧易永续合约。

## 核心功能
1. **多周期信号共振**：15m+1h+4h三周期MACD过滤，提升信号胜率
2. **动态风控**：ATR止盈止损+单笔2%风险控制+三重熔断保护
3. **全自动交易**：信号生成→风控检查→下单→成交确认→持仓同步
4. **历史回测**：基于30天历史数据验证策略胜率/收益
5. **日志监控**：按天轮转日志，记录所有交易/风控行为
